App de Finanzas Personales

Aplicación móvil desarrollada en Flutter para el control de ingresos, egresos y balances financieros, conectada a una API propia en Python (Flask) con base de datos SQL Server.

Descripción del Proyecto

Esta app permite gestionar de manera sencilla las finanzas personales desde cualquier dispositivo Android. Los usuarios pueden agregar movimientos, clasificarlos por categoría y obtener reportes visuales en tiempo real.

El backend fue desarrollado en Python con el framework Flask, utilizando autenticación JWT y conexión a SQL Server mediante pyodbc. El frontend móvil fue desarrollado en Flutter con manejo de estado mediante Provider.

Tecnologías Utilizadas

  • Frontend móvil: Flutter

  • Backend: Python (Flask, JWT, pyodbc)

  • Base de datos: SQL Server

  • Infraestructura: API REST con seguridad por tokens

Características Principales

  • Dashboard con resumen financiero mensual
  • Gráficos dinámicos de ingresos y egresos
  • Control de categorías personalizadas
  • Autenticación segura (JWT)
  • Conexión en tiempo real a la API
